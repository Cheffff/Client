Aide R.A Hololens

 

 

Mettre en route les hololens

Appuyer sur le bouton power pour allumer les HoloLens

 

Pour sortir de veille, appuyer pendant 1 seconde sur le bouton power, ou dite « hey Cortana »

Mettre en veille les hololens

 

Pour mettre les HoloLens appuyer sur le bouton power 1 seule fois, les lumières devrait clignoter.

Les hololens passe automatique en veille après 3 minutes d’inactivité, ou que la batterie atteint 10%.

Pour sortir de veille, appuyer pendant 1 seconde sur le bouton power, ou dite « hey Cortana »

 

 

Eteindre les HoloLens

Pour éteindre les HoloLens, appuyer sur le bouton power pendant 4 secondes jusqu’à a ce que les lumières s’éteignent

Les hololens s’éteignent automatiquement des que la batterie atteint 1%.

 

La zone de geste

Toutes les actions effectuées via les hololens doivent être fait dans un champ définit par camera des hololens.

Toutes action effectuée en dehors ne sont pas pris en compte.

 

 

Les gestes

4 types de gestes peuvent être faite via les hololens

Bloom

 

Le bloom permet d’ouvrir le menu start, il permet aussi de mettre en pause un jeu, ou réduire toutes les fenêtres.

Pour réaliser un  bloom, mettez votre main a plat, puis relevez vos doigt.

Tip

Vous pouvez sinon demandez a Cortana de revenir a l’accueil.

 

 

 

 

 

 

Gaze

 

 Le gaze correspond à la souris des hololens. Elle permet de placer un pointeur sur une fenêtre ou autre points visible via les hololens.

Pour utiliser le gaze, bougez votre tête de gauche à droite.

 

 

 

 

Air tap

L’air tap correspond à un clic de souris. Il s’utilise avec le gaze afin de provoquer une action.

  1. Pointé l’élément souhaité.
  2. Lever vote index et votre pouce.
  3. Bougez vos doigt vers le bas rapidement et reprenez la position de base.

Tap and hold

Le tap and hold est un glisser déposer, qui vous permet de déplacer des éléments. Tout comme le clic le mouvement est similaire, mais vous devez garder les doigts colle tout le long du déplacement

 

 

Connexion Wi-fi

Pour pouvoir utiliser les hololens en wifi vous devez vous connectez

  1. Bloom sur l’accueil, sélectionner Settings .
  2. Utiliser le gaze sur l’app Settings, et ‘air tap’ pour cliquer.
  3. Sélectionnez Network & Internet > Wi-Fi.
  1. Sélectionnez un a réseau puis cliquez sur > Connect.
  1. Tapez votre mot de passe, et cliquez sur Next.
